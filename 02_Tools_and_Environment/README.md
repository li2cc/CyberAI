# Module 2 — Tools & Environment

A 1-week setup module: students stand up the toolchain they'll use all semester (VS Code,
Python, Git/GitHub, Colab + Google Drive, API keys in a hidden `.env`/Colab Secrets) and prove
it works by running the Module 1 code in Colab **and** locally.

There is no lecture notebook here — the work is environment setup, guided by:
- **Slides:** `Module2_Tools.pptx` (in the instructor `assignments and projects/` folder).
- **Lab:** `Module2_Lab_and_Rubric.docx` — "Set Up & Verify Your Environment" (create your IT 7075
  GitHub repo, mount Drive in Colab, select the right Python interpreter, store keys in `.env`).

See the root [README](../README.md) for where this module sits in the course.
