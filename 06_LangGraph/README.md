# Module 6 — LangGraph (Stateful Agents)

A 2-week module based on **Omar Santos's** material (book Ch. 4; LinkedIn course Section 4; repo
`part5_agents_and_tools/agent_deep_dive/langgraph`). LangGraph extends LangChain to **stateful,
cyclical agent workflows** — nodes, edges, shared state, conditional routing, tools, and
human-in-the-loop — which is what real autonomous security agents need.

**Planned contents (same design as the other Omar modules):**
- `Module6_LangGraph.pptx` — lecture slides with speaker scripts.
- `langgraph_part1.ipynb` — concepts: state, nodes, edges, the graph; a simple security agent
  (e.g., a malicious-URL analyzer, after Omar's `basic_example.py`).
- `langgraph_part2.ipynb` — tools, conditional branching, memory, and a multi-step agent (after
  Omar's `ethical_hacking_agent.py` / `cisa_kev_agent.py`).
- `langgraph_sample.ipynb` — a complete worked sample-code notebook.
- Week-1 concept assignment + **Mini-Project 4** (a LangGraph agent for a security task).
